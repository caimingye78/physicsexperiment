摆动波实验讲义 · Pendulum Wave

视觉震撼、原理纯粹的高中物理演示。关键词：单摆周期、相位差、行波、再同步。

长度渐变的一排单摆，同时释放后某一瞬间连成的"行波"

一、实验原理

T = 2π√(L/g)　　→　　L = g · (T / 2π)²

设总循环时间 T_cycle（如 60 s），最长摆完成 base 次振动，其后每摆多 1 次：

Nₖ = base + (k−1)　；　Tₖ = T_cycle / Nₖ　；　Lₖ = g · (Tₖ / 2π)²

各摆相位逐渐错开形成行波，经过 T_cycle 后整数次振动完成，重新排成直线。

二、器材清单

  • 横杆约 70 cm（刚性好）+ 两个支架，离桌面 ≥ 40 cm
  • 15 个质量一致的小球（钢珠 / 大螺母）
  • 钓鱼线或细棉线、直尺、剪刀、记号笔
  • 释放杆（直尺/平板）、手机（慢动作录像）

四、操作步骤

  1. 按上表剪线、系球、等间距挂到横杆。
  2. 微调使每摆"悬挂点→质心"距离精确等于摆长值。
  3. 释放杆从下方托住所有球，对齐到同一水平线、同一角度（<15°）。
  4. 水平、干脆地平移抽走释放杆，使所有摆同时启动。
  5. 录像观察：行波 → 多段波 → 散乱 → 约 60 s 再同步。

五、可探究问题

  • 验证周期公式：数 30 s 内振动次数反推 T，与 2π√(L/g) 对比。
  • 测重力加速度：g = 4π²L / T²，与 9.8 比较并分析误差。
  • 再同步时间是否等于 60 s？讨论空气阻力、线伸长、释放误差。

六、安全提示

  • 风险极低，注意横杆/支架稳固。
  • 效果差多因摆长不准或释放不齐；摆角务必 < 15°。

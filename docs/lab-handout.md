# 摆动波实验讲义（Pendulum Wave）

> 一个视觉震撼、原理纯粹的高中物理演示实验。
> 关键词：单摆周期、相位差、行波、再同步。

---

## 一、实验目标

1. 观察一排长度渐变的单摆同时释放后形成的"行波—散乱—再同步"现象。
2. 验证单摆周期公式 `T = 2π√(L/g)`。
3. 由实测摆长与周期反推重力加速度 `g = 4π²L / T²`。

## 二、实验原理

单摆在小角度下做简谐运动，周期只与摆长有关：

```
T = 2π √(L / g)        =>        L = g · (T / 2π)²
```

设定一个**总循环时间** `T_cycle`（例如 60 秒）。让最长的摆在这段时间内完成
`base` 次全振动，其后每个摆依次多 1 次。于是第 k 个摆（k 从 1 开始）：

```
振动次数  N_k = base + (k − 1)
周期      T_k = T_cycle / N_k
摆长      L_k = g · (T_k / 2π)²
```

所有摆从同一直线出发，因周期略不同而逐渐错开相位，连成一条移动的波；
经过 `T_cycle` 后，每个摆都完成整数次振动，于是再次排成同一直线，循环往复。

## 五、操作步骤

1. 按上表剪好 15 段摆线，留出打结余量，逐一系上小球并挂到横杆等间距的点上。
2. 微调每个摆，使悬挂点到小球质心的距离精确等于"摆长"列数值。
3. 用释放杆从下方托住全部小球，使它们处在同一水平线、同一角度（< 15°）。
4. 水平、干脆地平移抽走释放杆，让所有摆**同时**开始运动。
5. 录像观察：行波 → 多段波 → 散乱 → 约 60 s 后再次同步。

## 六、可探究的问题

- **验证周期公式**：慢动作数某个摆 30 s 内的振动次数，反推 T，与 `2π√(L/g)` 对比。
- **测重力加速度**：用 `g = 4π²L / T²` 计算并与 9.8 比较，分析误差来源。
- **再同步时间**：实测循环周期是否等于设定的 60 s？讨论空气阻力、线伸长、释放不齐的影响。

## 七、安全与常见问题

- 风险极低；唯一注意点是横杆/支架要架稳，避免倾倒。
- 效果不好通常源于两点：摆长不够精确、释放不够整齐。多预留线长、设可微调结点。
- 摆角过大时简谐近似失效，波形会变乱，请控制在 15° 以内。

# 双缝干涉（Young's Double-Slit）实验讲义

> 重现 1801 年杨氏实验：用一束光证明"光是波"，并把经典考题中的 Δy = λL/d 亲手测出来。

## 实验目标
1. 观察相干光通过双缝后在屏上形成的明暗条纹，理解干涉的相长与相消条件。
2. 测量条纹间距，验证 `Δy ≈ λ·L / d`，并由此反推波长 λ。
3. 理解单缝衍射包络如何"调制"双缝干涉条纹的亮度，把两个经典题型合二为一。

## 实验原理
- 两条缝作为相干次级光源，到达屏上某点的光程差决定干涉结果。
- 双缝干涉强度分布：
  - `I = I0 * cos^2(pi*d*sinθ/λ)`
  - θ 为该点相对中央的方向角，d 为缝间距，λ 为波长。
- 实际每条缝有宽度 a，会产生单缝衍射，对干涉条纹形成包络调制：
  - `I = I0 * cos^2(pi*d*sinθ/λ) * sinc^2(pi*a*sinθ/λ)`
  - 其中 `sinc(u) = sin(u)/u`。
- 小角近似下 `sinθ ≈ y/L`，相邻亮纹间距：
  - `Δy ≈ λ*L / d`
- 相长（亮纹）条件：`d*sinθ = k*λ`，k = 0, ±1, ±2 …
- 相消（暗纹）条件：`d*sinθ = (k + 1/2)*λ`。
- 经典题对应：已知 d、L 和测得的 Δy，即可求 `λ = d*Δy / L`，这正是高考与竞赛常见的"测波长"题。

## 器材清单
- 激光笔（红 ~650 nm 或绿 ~532 nm）或单色钠灯。
- 双缝片（已知缝间距 d 与缝宽 a）。
- 白色观察屏 / 毫米刻度尺。
- 光具座、可调支架、卷尺（量屏距 L）。
- 可选：滤光片、白光光源（观察彩色条纹）。
- 本仿真页面（index.html）可离线演示，并支持白光模式。

## 操作步骤
1. 将激光、双缝片、屏依次固定在光具座上，使三者中心共线。
2. 让激光垂直照射双缝，在屏上找到清晰的明暗条纹。
3. 用卷尺量出双缝到屏的距离 L；用刻度尺量出相邻若干条亮纹的总宽度，除以条数得到 Δy。
4. 代入 `λ = d*Δy / L` 计算波长，与激光标称值比较。
5. 改变 L，观察 Δy 随之线性变化，作 Δy–L 图验证斜率为 λ/d。
6. 在仿真中分别拖动 d、a、λ、L 四个滑块，对照屏上条纹与下方强度曲线的变化；打开/关闭"单缝包络"观察调制效果，切换"白光模式"看彩色条纹。

## 可探究问题
1. 增大缝间距 d 时条纹是变密还是变疏？用公式 Δy = λL/d 解释。
2. 缝宽 a 改变时，整体条纹间距是否变化？被抑制（变暗）的是哪些条纹？为什么？
3. 当 d 是 a 的整数倍时，某些干涉亮纹恰好落在衍射暗纹处而"消失"（缺级），如何预测缺第几级？
4. 用红光与绿光分别实验，条纹间距哪个更大？说明 Δy 与 λ 的关系。
5. 白光照射时为什么中央是白色、两侧出现彩色？哪种颜色离中央更近？
6. 把缝宽 a 调到与 d 相近时，为什么只剩中央少数几条亮纹可见？

## 安全提示
- 切勿让激光直射眼睛，也不要对着他人或镜面反射；调光时视线保持在光束上方。
- 使用激光时在桌面铺设无强反光的暗色垫布。
- 钠灯等光源工作时温度较高，熄灭后待冷却再触碰。
- 实验室保持适度暗环境，但留有安全照明，避免绊倒。
- 仿真演示无激光风险，适合课堂大屏展示。

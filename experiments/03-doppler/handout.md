# 多普勒效应 实验讲义

> 一句话：声源与观察者发生相对运动时，接收到的频率会偏离声源发出的频率——靠近时变高，远离时变低。

## 实验目标
1. 观察运动声源发出的波前如何"前密后疏"，理解多普勒效应的几何成因。
2. 定量验证观测频率公式 f' = f x (v ± vo) / (v ∓ vs)。
3. 认识当声源速度超过波速时形成的马赫锥（声爆）现象。
4. 把"救护车鸣笛由尖变闷"这类生活/考试经典题，转化为可定量观测的实验。

## 实验原理
声源以速度 vs 运动时，每隔一个周期 T = 1/f 发出一个球面波前。由于声源在两次发射之间移动了一段距离，前方相邻波前被"压缩"、波长变短，后方波前被"拉长"、波长变长。

- 声速记为 v，声源发射频率记为 f，观察者运动速度记为 vo。
- 通用公式（沿连线方向）：f' = f x (v ± vo) / (v ∓ vs)
  - 观察者朝声源运动，分子取 (v + vo)；远离取 (v - vo)。
  - 声源朝观察者运动，分母取 (v - vs)；远离取 (v + vs)。
- 本模拟中观察者静止，vo = 0：
  - 前方（靠近）：f_front = f x v / (v - vs)
  - 后方（远离）：f_back  = f x v / (v + vs)
- 马赫数 M = vs / v。当 M ≥ 1 时，波前无法领先于声源，所有波前的切线包络成一个圆锥面（马赫锥），半顶角 θ 满足 sin θ = v / vs，即 sin θ = 1 / M。

## 操作步骤
1. 打开 index.html，先把"声源速度 vs"设为 0，观察波前是同心圆，前后频率相等。
2. 逐步增大 vs，观察波前向运动方向挤压，记录"前方/后方观测频率"的变化趋势。
3. 固定 v = 340 m/s、f = 440 Hz，分别取 vs = 0、60、120、200 m/s，把读数填入表格并与公式计算值对比。
4. 显示"固定观察者"光点，注意声源经过观察者瞬间，接收频率由高跳到低（音调骤降）。
5. 把 vs 调到大于 v，观察马赫锥出现，记录此时半顶角并用 sin θ = 1/M 验证。

## 可探究问题
1. 经典题实验化：一列火车以 vs 鸣笛 f 驶来又驶离，求站台听到的两段频率之差 Δf。用模拟读数验证 Δf = f x v x [1/(v - vs) - 1/(v + vs)]。
2. 若让观察者也运动（设想题），公式分子如何变化？为何"声源运动"与"观察者运动"即使相对速度相同，结果也不完全相等？
3. 已知前后两段观测频率，能否反推声源速度 vs 与发射频率 f？请推导反解公式。
4. 马赫锥半顶角随速度如何变化？战斗机从 1.2 倍声速加速到 2 倍声速，锥角变大还是变小？
5. 雷达测速、医院多普勒超声测血流，与本实验是同一原理吗？差别在哪？

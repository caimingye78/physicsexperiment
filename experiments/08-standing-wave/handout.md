# 弦上驻波（Standing Waves on a String）实验讲义

> 把"两端固定弦的谐振频率"这道经典题，变成一根能拨、能调、能看见节点的虚拟琴弦。

## 实验目标
1. 理解驻波是两列等幅、反向传播行波叠加的结果。
2. 认识节点（不动点）与波腹（振幅最大点）的位置规律。
3. 掌握两端固定弦的谐波条件：波长 λ = 2L/n，频率 f_n = n·v/(2L)。
4. 把"弦乐器为何只发出某些音高"这一现象与整数谐波联系起来。

## 实验原理
- 两端固定的弦上，一列右行波与一列左行波叠加：
  - `A*sin(k*x - w*t) + A*sin(k*x + w*t) = 2A*sin(k*x)*cos(w*t)`
- 因此驻波位移可写为：
  - `y(x,t) = 2A*sin(n*pi*x/L)*cos(w*t)`
  - 其中 L 为弦长，n 为谐波数（正整数），k = n*pi/L，w = k*v。
- **节点**满足 sin(k*x)=0，即 `x = i*L/n`（i = 0,1,...,n），共 n+1 个，始终不动。
- **波腹**位于相邻节点正中，`x = (i+0.5)*L/n`，共 n 个，振幅达到 2A。
- 两端固定（边界位移为零）要求弦长是半波长的整数倍：
  - `L = n*(λ/2)`，于是 `λ = 2L/n`。
- 由 v = λ*f 得到允许的谐振频率：
  - `f_n = v/λ = n*v/(2L) = n*f_1`，即基频 f_1 的整数倍。
- 经典对应题：吉他/小提琴弦定音、声管驻波、共振柱测声速，都是同一套谐波条件的应用。

## 器材清单
- 一根可张紧的弦（橡皮弦、尼龙线或细钢丝），两端固定在支架上。
- 弦振动发生器（机械振子）+ 信号发生器，或用手拨动。
- 砝码 + 滑轮用于改变张力 T（从而改变波速 v = sqrt(T/μ)）。
- 米尺、频率计、闪频仪（便于"定格"观察驻波）。
- 本仿真页面（index.html），可离线演示、切换谐波、拆解出两列行波。

## 操作步骤
1. 把弦两端固定，测量弦长 L，挂上砝码确定张力 T。
2. 打开振动发生器，从低频开始缓慢增大频率，直到弦出现稳定的单段驻波（基频 n=1）。
3. 记录此频率 f_1，并数出节点与波腹个数。
4. 继续升高频率，依次找到 n=2,3,4… 的谐波，记录每个 f_n 与对应的波腹数。
5. 验证 f_n 是否近似为 n·f_1，并用 λ = 2L/n 计算波长。
6. 在仿真中拖动 n、A、v 滑块复现各谐波；点击"显示行波分量"观察两列行波如何叠加成驻波；点击频谱柱或"拨弦一下"快速切换与激发。

## 可探究问题
1. 为什么两端固定时只能存在 λ = 2L/n 这些离散波长？把边界条件代入公式说明。
2. 改变张力（砝码）使波速 v 增大，各谐波频率如何变化？用 f_n = n*v/(2L) 预测并验证。
3. 节点真的完全不动吗？用仿真的"行波分量"解释为什么在节点处两列行波始终反相抵消。
4. 同一根弦上，n=2 的频率恰好是 n=1 的两倍，这与音乐里的"八度"有什么关系？
5. 如果把弦的一端改为自由端，谐波条件会变成什么（λ = 4L/(2n-1)）？为什么？
6. 弦的线密度 μ 增大（更粗的弦）时，相同张力下基频如何变化？

## 安全提示
- 张紧的弦或细钢丝若断裂会高速弹回，务必戴护目镜，且不要正对弦的延长线方向站立。
- 砝码下方不要放手或脚，防止脱落砸伤。
- 闪频仪频闪可能引起不适，光敏体质者慎用。
- 机械振子长时间运行注意发热与固定，避免支架松动。
- 仿真演示无上述风险，建议先在仿真中预估各谐波频率，再进行实物测量。

弦上驻波 Standing Waves on a String

两端固定的弦上，两列反向传播的行波叠加成驻波——节点不动，波腹剧烈起伏，只有特定谐波频率能稳定存在。

节点（不动） 波腹（振幅最大） 合成驻波 右行波 左行波
谐波频谱：当前谐波 n 对应的频率 f_n = n·v/(2L)，点击柱子可直接切换到该谐波。
乐音音阶（弦长不变改 n）： 基频 八度 十二度 双八度
驻波：
y(x,t) = 2A·sin(n·π·x/L)·cos(ω·t)

由两列行波叠加：
A·sin(kx−ωt) + A·sin(kx+ωt)

波长 λ = 2L/n
频率 f_n = n·v/(2L)
节点数 = n+1，波腹数 = n。

# 气体动理论 实验手册

> 微观小球的碰撞，如何变成宏观的温度与压强？本实验用上百个弹性小球，亲眼见证麦克斯韦速率分布的诞生。

## 实验目标

1. 理解理想气体的微观模型：大量做无规则热运动、彼此弹性碰撞的小球。
2. 观察并解释速率分布如何自发趋向麦克斯韦—玻尔兹曼分布。
3. 通过改变粒子数 N、温度 T、体积 V，定性验证理想气体状态方程 P·V = N·k·T。
4. 建立"温度是分子平均平动动能的量度"这一核心观念。

## 实验原理

- 微观模型：气体由大量小球组成，小球之间、小球与器壁之间均发生完全弹性碰撞，碰撞间隔做匀速直线运动。
- 温度与动能：温度越高，分子平均速率越大。二维理想气体满足 (1/2) m <v²> = k·T；三维则为 (1/2) m <v²> = (3/2) k·T。
- 压强来源：气体压强来自大量分子持续撞击器壁的平均效果。单位时间、单位长度器壁上接收到的分子动量变化，即为压强 P。
- 状态方程：综合上述关系可得 P·V = N·k·T。式中 P 为压强、V 为体积、N 为分子数、k 为玻尔兹曼常数、T 为热力学温度。
- 麦克斯韦分布：处于热平衡的气体，分子速率服从确定的统计分布。二维情形下 f(v) 正比于 v·exp(−m·v²/2kT)，存在一个最概然速率（分布峰值）。即使初始时所有小球速率相同，弹性碰撞也会在很短时间内把分布"搅"成该形状。

## 器材清单

本实验为纯软件互动模拟，无需实物器材。运行环境：

- 任意现代浏览器（Chrome / Edge / Firefox / Safari 均可）。
- 直接双击 index.html 即可离线运行，无需联网、无需安装任何插件。
- 建议屏幕分辨率不低于 1280×720，以便同时观察粒子盒与速率直方图。

## 操作步骤

1. 打开 index.html，观察盒内小球的无规则运动。下方蓝色直方图为实时速率分布，黄色曲线为对应温度的麦克斯韦理论分布。
2. 静置约十秒，观察直方图如何逐渐贴合黄色理论曲线——这就是热平衡的建立过程。
3. 拖动"温度 T"滑块调高温度，观察小球整体变快、颜色偏橙，直方图峰值右移、变宽。
4. 点击"瞬间加热 / 瞬间冷却"按钮，观察分布曲线的快速平移与回稳。
5. 拖动"体积 V"滑块移动右侧活塞压缩气体，观察"压强 P"读数上升（撞墙更频繁）。
6. 改变"粒子数 N"，配合不同 T、V，记录右侧 P·V/(N·T) 读数，验证其在各种组合下近似为同一常数。

## 可探究问题

1. 初始时把所有小球设为相同速率，分布需要多久才稳定到麦克斯韦形状？这个时间与粒子数有关吗？
2. 保持温度不变、压缩体积到一半，压强大约变为原来的几倍？是否符合 P 与 V 成反比（玻意耳定律）？
3. 保持体积不变、把温度提高一倍，压强如何变化？是否符合 P 与 T 成正比？
4. 最概然速率、平均速率、均方根速率三者大小关系如何？尝试从直方图上估读最概然速率随温度的变化趋势。
5. 如果让一半小球质量加倍（思考实验），平衡时它们的平均速率会更大还是更小？平均动能呢？

## 安全提示

- 本实验为软件模拟，无电气、机械或化学风险。
- 长时间盯屏请遵循用眼卫生，每 20 分钟远眺休息。
- 粒子数过大可能使较老设备卡顿，若动画不流畅请适当调低 N。
- 模拟采用约化单位（令 m = 1、k = 1）以便观感，读数用于定性与趋势分析，不代表真实物理量的精确数值。

# 受迫振动与共振（Driven Damped Resonance）实验讲义

> 把高考/竞赛常见的"共振"问答题变成一台看得见振幅飙升的仿真机。

## 实验目标
1. 理解受迫振动的稳态：振子最终以**策动力的频率** ω 振动，而非自身固有频率 ω₀。
2. 观察并绘制共振曲线（振幅 A 对策动频率 ω），找出共振峰的位置与高度。
3. 研究阻尼系数 β 对共振峰尖锐程度（品质因数 Q）的影响。
4. 把"塔科马大桥、高脚杯破碎、荡秋千"等经典案例与公式定量联系起来。

## 实验原理
- 受迫阻尼振子的运动方程：
  - `x'' + 2*beta*x' + w0^2*x = F*cos(w*t)`
  - 其中 w0 为固有角频率，beta 为阻尼系数，F 为单位质量上的策动力幅，w 为策动角频率。
- 经过暂态后进入稳态，振子按策动频率振动，稳态振幅为：
  - `A = F / sqrt((w0^2 - w^2)^2 + (2*beta*w)^2)`
- 位移落后于策动力一个相位：
  - `phi = atan2(2*beta*w, w0^2 - w^2)`
- 共振（振幅最大）发生在：
  - `w_r = sqrt(w0^2 - 2*beta^2)`（阻尼很小时近似为 w_r ≈ w0）。
- 品质因数描述峰的尖锐程度：`Q = w0 / (2*beta)`，Q 越大峰越高越窄、越"危险"。
- 能量角度：当 w ≈ w0 时，策动力每个周期都在"正确的时刻"做正功，能量持续累积，振幅大幅增大。

## 器材清单
- 弹簧振子（弹簧 + 质量块）一套，或单摆作为替代。
- 可调频率的机械激励器（如偏心电机 / 信号发生器驱动的电磁线圈）。
- 位移传感器或带刻度的标尺 + 手机慢动作摄像。
- 阻尼装置（可在水/油中改变浸入深度来调节 β）。
- 秒表、频率计。
- 本仿真页面（index.html），无设备时离线演示，并可一键复现三个经典案例。

## 操作步骤
1. 先不加策动力，让振子自由振动，测出固有周期 T0，算出 `w0 = 2*pi/T0`。
2. 接通激励器，从很低的频率开始，每隔一小步增大策动频率 ω，等待暂态消失后读取稳态振幅 A。
3. 记录多组 (ω, A)，在坐标纸上描点画出共振曲线。
4. 找到振幅最大的策动频率，与理论值 `w_r = sqrt(w0^2 - 2*beta^2)` 比较。
5. 改变阻尼（调节浸入深度），重复上述步骤，比较不同 β 下峰的高度与宽度。
6. 在仿真中拖动 ω、ω₀、β、F 滑块复现实验，点击"一键调到共振"观察振幅爆发；点击案例标签体会真实场景。

## 可探究问题
1. 稳态时振子到底以 ω 还是 ω₀ 振动？用仿真的实时 x(t) 曲线验证。
2. 共振峰一定恰好在 ω = ω₀ 吗？阻尼增大时峰会向哪边移动？用 `w_r = sqrt(w0^2 - 2*beta^2)` 解释。
3. 把振幅最大值 A_max 与 1/β 作图，是否近似成正比？这说明阻尼对共振的什么作用？
4. 品质因数 Q = w0/(2*beta) 与峰的"半高宽"有什么定量关系？
5. 塔科马大桥为何会因风而毁？请用"涡街频率接近固有频率 + 低阻尼"来定量讨论。
6. 荡秋千时"在最低点附近蹬腿"对应公式中的什么条件（频率匹配与相位）？

## 安全提示
- 共振时振幅可能远超预期，机械装置要预留足够空间并固定底座，防止飞出伤人。
- 高脚杯/玻璃类破碎演示需戴护目镜，并在封闭透明罩内进行。
- 大功率激励器通电部分注意绝缘，避免漏电。
- 长时间高频运行电机会发热，注意散热与触碰防护。
- 仿真演示无上述风险，推荐先在仿真中预测危险频率，再决定是否进行实物实验。

# 李萨如图形 实验讲义

> 一句话：两个互相垂直的简谐振动叠加，光点会在屏幕上画出由频率比和相位差决定的封闭曲线。

## 实验目标
1. 理解李萨如图形是"两个垂直方向简谐振动合成"的结果。
2. 掌握用图形相切点数判定频率比 a : b 的方法。
3. 观察相位差 δ 如何改变同一频率比下曲线的姿态（直线、椭圆、圆、斜环）。
4. 把"示波器比较音叉频率"这一经典实验题，转化为可亲手调节的互动模拟。

## 实验原理
设水平、竖直方向的振动分别为：
- x = A x sin(a x t + δ)
- y = B x sin(b x t)

其中 a、b 为两方向的角频率（这里用整数比代表频率比），δ 为相位差。

- 当 a : b 为简单整数比时，光点经过有限时间后回到起点，轨迹为封闭曲线。
- 判频率比法则：曲线与一条水平线的切点数记为 Nx，与一条竖直线的切点数记为 Ny，则 频率比 a : b = Ny : Nx（即"水平频率 : 竖直频率 = 竖直切点数 : 水平切点数"）。
- 特例：a : b = 1 : 1 时，随 δ 由 0 变到 90 度，图形从一条 45 度直线变为正椭圆再变为圆。
- 应用：示波器把待测信号接 X 偏转板、已知标准信号接 Y 偏转板，根据出现的李萨如图形即可读出两者频率比，从而测出未知频率。

## 器材清单
- 可运行本页 index.html 的电脑或平板（无需联网）。
- 选做实物对照：双通道示波器、两台信号发生器（或两支不同频率音叉 + 话筒）。
- 选做：附反射镜的两支音叉与激光笔，可在墙上投出李萨如光迹。

## 操作步骤
1. 打开 index.html，先用预设按钮依次查看 1:1、1:2、2:3、3:4 四种图形。
2. 固定 a = 1、b = 1，缓慢拖动相位差 δ 从 0 到 360 度，记录直线 → 椭圆 → 圆 → 椭圆的变化。
3. 选定一个比例（如 3:2），数出图形与水平边、竖直边的相切点数，验证 a : b 与切点数之比的关系。
4. 调整"拖尾保留"，对比短余晖（看清光点运动）与长余晖（看清完整曲线）的效果。
5. 打开"相位自动旋转"，观察图形像三维曲面被连续翻转般演化，体会相位差的几何意义。

## 可探究问题
1. 经典题实验化：示波器 Y 接 50 Hz 标准信号，X 接未知音叉，屏上出现稳定的 2:3 图形，求音叉频率（在模拟中反推验证）。
2. 为什么频率比为无理数时图形永不闭合？这与"周期是否可公度"有什么关系？
3. 同一频率比下，δ = 0 与 δ = 180 度的图形看起来一样吗？请解释差异来自何处。
4. 如果两个振动振幅不同（A ≠ B），图形会如何被"压扁"？这会影响切点计数法吗？
5. 把李萨如法与"拍频法"比较：测量两个相近频率时，哪种更灵敏？为什么？

## 安全提示
- 本模拟为软件演示，无电气或机械危险。
- 使用真实示波器时注意高压探头与接地，避免短路。
- 进行激光投影实验时，严禁激光直射眼睛，使用低功率（≤1 mW）激光笔。
- 长时间注视高亮度屏幕注意适当休息，保护视力。

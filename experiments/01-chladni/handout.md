# 克拉尼图形（Chladni Plate）实验讲义

> 把抽象的"二维驻波"变成看得见、摸得着的沙画。

## 实验目标
1. 观察振动方板上沙粒聚集形成的节线图案，理解"节点处不振动"的物理含义。
2. 建立模式数 (m, n) 与图案复杂度、振动频率之间的对应关系。
3. 把"边界条件 + 驻波"这一经典物理题转化为可视化实验，体会从公式到图形的过程。

## 实验原理
- 方板被驱动振动时，板面各点的位移构成二维驻波。位移为零的曲线称为**节线**，沙粒会从剧烈振动区被"抖"到节线上堆积。
- 自由方板的一种常见驻波场可写为两个本征模式的反对称组合：
  - `f(x,y) = cos(m*pi*x)*cos(n*pi*y) - cos(n*pi*x)*cos(m*pi*y)`
  - 其中 x, y 取单位方板坐标（0 到 1），m、n 为整数模式数。
- 节线即满足 `|f(x,y)| ≈ 0` 的位置；沙粒最终沿这些曲线分布。
- 模式的相对频率近似正比于 `sqrt(m^2 + n^2)`，因此 (m, n) 越大 → 频率越高 → 节线越密、图案越细。
- 经典对应题：弦的驻波 `y = A*sin(n*pi*x/L)` 是一维情形；本实验是它在二维方板上的推广。

## 器材清单
- 金属方板（黄铜或钢板）一块，中心用螺栓固定。
- 信号发生器 + 功率放大器 + 振动激励器（或小提琴弓拉板边）。
- 细沙或盐粒若干。
- 频率计 / 示波器（用于读取驱动频率）。
- 防护眼镜、耳塞（高频噪声防护）。
- 本仿真页面（index.html）可在无设备时离线替代演示。

## 操作步骤
1. 将方板水平固定，中心点用螺栓锁紧到激励器上。
2. 在板面均匀撒一层薄沙。
3. 从低频开始缓慢增大驱动频率，观察沙粒何时突然"跳动并重新聚集"——此时达到一个共振模式。
4. 记录该频率，并用相机拍下图案，标注估计的 (m, n)。
5. 继续升高频率，找出尽可能多的模式，比较图案复杂度与频率的关系。
6. 在仿真中拖动 m、n 滑块复现真实图案，用"对比度/沙量"滑块调节聚集效果，用"撒沙/重振"重新分布沙粒。

## 可探究问题
1. 为什么 (m, n) 与 (n, m) 的图案互为镜像？这与公式的反对称结构有什么关系？
2. 频率与 `sqrt(m^2 + n^2)` 是否近似成正比？用实测数据作图验证。
3. 改变固定点的位置（如固定边缘而非中心）会怎样改变可激发的模式？
4. 沙粒堆积到节线，盐粒（更轻）却可能聚到波腹（Faraday 效应）——为什么介质和频率会影响聚集位置？
5. 把一维弦驻波公式 `y = A*sin(n*pi*x/L)` 与二维场公式对比，说明节点从"点"变成"线"的几何升维。

## 安全提示
- 大功率激励器与高频声音可能损伤听力，请佩戴耳塞并控制音量。
- 金属板边缘锋利，操作时戴手套，避免划伤。
- 共振时板边振幅大，勿用手直接按压正在振动的板面。
- 撒沙、清理时佩戴护目镜，防止沙粒进入眼睛。
- 仿真演示无上述风险，可放心在课堂投影使用。

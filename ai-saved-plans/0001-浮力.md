### 实验名称：自制“潜水员”——探究浮力与液体密度及排开液体体积的关系

### 实验目的
1.  通过自制“潜水员”模型，直观感受物体浮沉状态的变化。
2.  探究影响浮力大小的两个关键因素：液体的密度和物体排开液体的体积。
3.  学习并应用控制变量法进行科学探究。
4.  培养小组合作、动手操作和数据分析的能力。

### 涉及物理原理
1.  **阿基米德原理：** 浸在液体中的物体受到的浮力，等于它排开的液体所受的重力。公式：**F浮 = ρ液 g V排**。
2.  **物体浮沉条件：** 当 **F浮 > G物** 时，物体上浮；**F浮 = G物** 时，物体悬浮；**F浮 < G物** 时，物体下沉。

### 创新点说明
1.  **可视化与可操控性：** 传统实验多用弹簧测力计测浮力，数据抽象。本实验通过“潜水员”的沉浮状态，将“浮力变化”转化为“视觉可观察的深度变化”，更直观、有趣。
2.  **低成本与高互动：** 核心器材是塑料滴管和回形针，成本极低，人人可得。实验过程需要小组不断调整“潜水员”的配重和外界压力，互动性强。
3.  **定量比较的引入：** 传统“潜水员”实验多只做定性观察。本方案创新性地引入 **“盐度计”** （简易密度计）和 **“水位标记”** ，让学生能定量比较不同液体密度和排开液体体积对浮力的影响，从“看现象”进阶到“比数据”。
4.  **控制变量法的完美载体：** 通过改变盐水的浓度（控制V排不变）和改变“潜水员”浸入水中的深度（控制ρ液不变），清晰展示了两个变量对浮力的影响。

### 实验步骤
**实验一：探究浮力与液体密度的关系（控制V排不变）**

1.  **制作“潜水员”：** 将滴管胶帽吸满空气，用回形针缠绕在滴管尾部，调整回形针数量，使滴管能**竖直悬浮**在水中，且大部分体积浸没，仅胶帽顶端露出水面。记录此时回形针数量。
2.  **准备“潜水舱”：** 将大塑料瓶装满清水，放入“潜水员”，拧紧瓶盖。
3.  **建立基准：** 用手挤压瓶身，观察“潜水员”下沉。记录当“潜水员”完全浸没在某一固定深度（例如瓶身中央）时，瓶内清水的水位线（用记号笔在瓶身上标记）。
4.  **改变密度：** 将“潜水员”取出，倒掉瓶中一半的水。加入**50克**食盐，搅拌完全溶解，制成盐水。再将“潜水员”放回瓶中，加满盐水至瓶口，拧紧瓶盖。
5.  **观察与测量：** 再次用手挤压瓶身，使“潜水员”下沉到**与步骤3相同的深度**（即对准之前的水位标记）。记录此时瓶身被挤压的程度（可用手指感觉的“松紧度”定性描述，或更精确地用“挤压后瓶身周长变化”定量描述，但简化为“挤压程度”即可）。
6.  **重复：** 再向瓶中加50克食盐（共100克），重复步骤5。比较三次实验中，使“潜水员”到达同一深度所需的挤压程度。

**实验二：探究浮力与排开液体体积的关系（控制ρ液不变）**

1.  **恢复清水：** 将瓶中盐水倒掉，重新装入清水，放入同一“潜水员”。
2.  **建立不同深度：** 在瓶身上用记号笔标出三个不同的深度标记：A（较浅）、B（中间）、C（较深）。
3.  **实验操作：** 挤压瓶身，使“潜水员”依次下沉到A、B、C三个深度。记录每次到达该深度时，瓶身的**挤压程度**。
4.  **数据记录：** 思考：“潜水员”在不同深度时，它排开水的体积（V排）是否相同？（提示：在悬浮状态下，V排 = 潜水员自身的体积，是固定的。但当我们挤压瓶身时，水压增大，滴管胶帽被压缩，导致“潜水员”自身体积减小，从而V排减小，浮力减小而下沉。所以，不同深度对应的是不同的V排。）

### 涉及物理原理
1.  **阿基米德原理：** 浸在液体中的物体受到的浮力，等于它排开的液体所受的重力。公式：**F浮 = ρ液 g V排**。
2.  **物体浮沉条件：** 当 **F浮 > G物** 时，物体上浮；**F浮 = G物** 时，物体悬浮；**F浮 < G物** 时，物体下沉。

### 创新点说明
1.  **可视化与可操控性：** 传统实验多用弹簧测力计测浮力，数据抽象。本实验通过“潜水员”的沉浮状态，将“浮力变化”转化为“视觉可观察的深度变化”，更直观、有趣。
2.  **低成本与高互动：** 核心器材是塑料滴管和回形针，成本极低，人人可得。实验过程需要小组不断调整“潜水员”的配重和外界压力，互动性强。
3.  **定量比较的引入：** 传统“潜水员”实验多只做定性观察。本方案创新性地引入 **“盐度计”** （简易密度计）和 **“水位标记”** ，让学生能定量比较不同液体密度和排开液体体积对浮力的影响，从“看现象”进阶到“比数据”。

1.  **阿基米德原理：** 浸在液体中的物体受到的浮力，等于它排开的液体所受的重力。公式：**F浮 = ρ液 g V排**。
2.  **物体浮沉条件：** 当 **F浮 > G物** 时，物体上浮；**F浮 = G物** 时，物体悬浮；**F浮 < G物** 时，物体下沉。
3.  **核心原理（修正）：** 挤压瓶身→水压增大→“潜水员”胶帽内空气被压缩→“潜水员”自身体积减小（V排减小）→平均密度增大→下沉。反之亦然。

### 创新点说明
1.  **科学性与严谨性提升：** 修正了原实验二中关于“下沉深度”与“V排”关系的错误解释，将自变量明确为“对瓶身的挤压程度”，使物理过程清晰、逻辑自洽。
2.  **定量测量客观化：** **引入“压力传感器”或“弹簧测力计”** 替代主观的“挤压程度”判断，将定性描述升级为定量数据（力的大小），显著提升实验的科学性和数据的可靠性。
3.  **控制变量法的精准应用：** 实验二创新性地设计为“在**同一深度**，改变挤压程度”，完美控制了“水压”这一潜在干扰变量，让学生清晰看到V排变化对浮力的直接影响。
4.  **“挑战性”任务驱动：** 增设“未知液体密度探测”环节，将知识从验证推向应用，培养学生的逆向思维和综合解决问题的能力，提升了实验的深度和趣味性。

### 实验步骤
**实验一：探究浮力与液体密度的关系（控制V排不变）**

1.  **制作“潜水员”：** 将滴管胶帽吸满空气，用回形针缠绕在滴管尾部，调整回形针数量，使滴管能**竖直悬浮**在水中，且大部分体积浸没，仅胶帽顶端露出水面。记录此时回形针数量。
2.  **准备“潜水舱”：** 将大塑料瓶装满清水，放入“潜水员”，拧紧瓶盖。
3.  **建立基准：** 在瓶身中央标记一个固定深度线。用手轻轻挤压瓶身，使“潜水员”下沉，直至其胶帽顶端恰好对准该深度线。记录此时**压力传感器（或弹簧测力计）的读数**，作为基准值。
4.  **改变密度：** 将“潜水员”取出，倒掉瓶中一半的水。加入**50克**食盐，搅拌完全溶解，制成盐水。再将“潜水员”放回瓶中，加满盐水至瓶口，拧紧瓶盖。
5.  **观察与测量：** 再次用手挤压瓶身，使“潜水员”下沉到**与步骤3相同的深度**（对准深度线）。记录此时**压力传感器的读数**。比较与清水时的读数差异。
6.  **重复：** 再向瓶中加50克食盐（共100克），重复步骤5。比较三次实验中，使“潜水员”到达同一深度所需的压力值。
7.  **分析：** 液体密度越大，使“潜水员”到达同一深度所需压力值越**大**（还是**小**？），说明浮力越**大**（还是**小**）？为什么？

**实验二：探究浮力与排开液体体积的关系（控制ρ液不变）**

1.  **恢复清水：** 将瓶中盐水倒掉，重新装入清水，放入同一“潜水员”。
2.  **固定深度：** 在瓶身中央标记一个固定深度线（与实验一相同）。
3.  **改变V排：** 保持瓶内为清水。操作员挤压瓶身，使“潜水员”下沉并**恰好悬浮在步骤2标记的固定深度**。
4.  **分档测量：** 在保持“潜水员”悬浮于该固定深度的前提下，改变挤压瓶身的力度，分为“轻”、“中”、“重”三档。记录每一档对应的**压力传感器读数**，并同时用肉眼观察并记录“潜水员”胶帽的**压缩程度**（可用“微小”、“明显”、“显著”定性描述，或用直尺测量胶帽凹陷深度定量描述）。
5.  **思考：** 当压力增大时，胶帽被压缩，V排是变大了还是变小了？为了在相同深度悬浮，浮力必须等于重力（重力不变），所以浮力不变。那V排变小了，浮力不变，这说明了什么？说明液体密度（清水）不变时，V排越小，浮力越**小**？**（注意：此处逻辑是，我们通过增大压力来减小V排，从而让浮力减小到与重力相等，实现悬浮。改变的是V排，而不是浮力。记录数据时应关注：压力越大 → V排越小 → 浮力越小）**。
    *   **教师引导：** 可以引导学生记录“压力值”和“胶帽压缩程度”，并分析两者关系。

### 安全注意事项
1.  **防止水溅：** 实验过程中挤压瓶身时，瓶口可能漏水，请在桌面铺上毛巾或在水盆上方操作。
2.  **轻拿轻放：** 塑料瓶和滴管都是易碎品，避免摔落或用力过猛导致破裂。
3.  **食盐勿入口：** 实验用的食盐可能被污染，禁止食用。
4.  **工具归位：** 实验结束后，将所有器材清洗干净并放回原位。
